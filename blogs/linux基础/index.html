常用命令

  1. ls /：展示当前目录下的所有文件

    1. etc：存放配置文件和子目录
    2. var：存放日志以及线程程序
    3. bin/usr/sbin：系统用户使用的命令

  2. 对文件的操作

    1. cd ..：进入上一级目录
    2. cd ../：进入根目录
    3. cd **：进入任意文件夹
    4. ls：展示所有文件夹下的子文件夹和子文件
    5. ls -l：展示文件属性以及文件所属的用户和组

  3. l rwx rwx rwx

    0 123 456 789

    下标为0：代表确定文件类型

    下标为123：确定文件的所属 拥有该文件的权限

    下标为456：确定属性组 所有有权限的用户组

    下标为789：其他用户拥有该文件的权限

    1. 下标为0位置还可以是：- d l b c

      -：是一个文件

      d：是一个目录

      l：是链接文档

      b：是装置文件，可提供存储的接口设备

      c：是串行接口设备

    2. 后3位制

      r：读

      w：写

      x：执行

      -：权限

  4. 屏幕清空：clear/ctrl+l

  5. 更改文件属性：chgrp：更改文件的属组

  6. 创建文件夹：mkdir **

  7. 删除文件夹：rmdir **

  8. 显示目前的目录：pwd

  9. 将文件从一处复制到另一处：cp

  10. 移动/重命名：mv

    1. -f：文件已经存在直接覆盖
    2. -i：询问是否覆盖
    3. -r：文件升级

  11. 移除：rm

    1. -f：忽略不存在的文件
    2. -i：互动模式
    3. -r：递归删除

  12. 编辑文件：vi

    1. i：insert输入模式
    2. :q：退出
    3. :q!：退出不报存
    4. :w：保存
    5. : wq：保存并退出
    6. :x：保存并退出
    7. 命令模式 command mode
    8. 输入模式 insert mode
    9. 底线模式 lastline mode

  13. 查看文件内容

    1. cat：由第一行显示内容
      1. cat -a
      2. cat -b：显示行号
      3. cat -e
      4. cat -n：显示行号，空行也占一行
      5. cat -v：显示特殊字符
    2. tail：查看文件后边几行
      1. -n：
      2. -f：
    3. less
      1. f：下一页
      2. 回车：换行
      3. q：退出

  14. 添加用户账户：useradd ** 用户名

    1. -g：用户组
    2. -u：用户号
    3. -d：指定用户主目录
    4. -s：shell文件 指定用户登录sell

    2. 编辑创建的好的文件
