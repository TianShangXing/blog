MySQL

MySQL 是最流行的关系型数据库管理系统，在 WEB 应用方面 MySQL 是最好的 RDBMS(Relational Database Management System：关系数据库管理系统)应用软件之一。

官网： https://www.mysql.com/cn/

数据库迁移

建立数据库映射关系
python manage.py makemigrations myapp # 子应用名称

建表
python manage.py migrate
